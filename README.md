## Syntax available
Available files: 
- LTPA_PREDIMED_making_data.R: elaboration of study databases
- LTPA_PREDIMED_2cat.R: study analyses using MVLTPA categorized as a binary variable (<100 vs ≥100 METs-min/d)
- LTPA_PREDIMED_4cat.R: study analyses using MVLTPA categorized into 4 levels (0, <100, ≥100-<200, ≥200 METs-min/d)
